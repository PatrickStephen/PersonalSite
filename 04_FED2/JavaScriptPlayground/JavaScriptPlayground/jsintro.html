Functions

Functions can be written in JS and called upon in reaction to an event. Allow the user to enter a color. When they click a button (event), call a function that will display a message along with their input.

Water Weight

Assume a gallon of water weighs 8.33lbs. Allow a user to enter a number of gallons. Upon clicking a button, call a function to calculate the total weight and display that value.

Loops

Looping also exists in JS. Here, we use a for loop to display 1-10.

Single Milestone

Allow a user to enter thier age. When they click a button, call a function that uses a switch to determine if there are any milestones for them.

Multiple Milestones

Allows a user to enter their age. WHen they click a button, call a function that uses if/else to display all milestone that they have reached.
